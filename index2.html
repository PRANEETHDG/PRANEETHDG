    HTML


    HTMLHyperText Markup Language is the standard language for creating web pages. HTML describes the structure of a web page using markup. HTML elements are the building blocks of HTML pages.

    
        <!DOCTYPE html>
        <html>
        <head>
            <meta charset="UTF-8">
            <title>My Webpage</title>
        </head>
        <body>
        
        <h1>Hello, World!</h1>
        <p>This is an example of HTML code displayed on a webpage.</p>
        
        </body>
        </html>
    The <mark> tag highlights text, while <del> and <ins> tags show deleted and inserted text, respectively.
